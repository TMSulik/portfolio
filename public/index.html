About Me

Tom Sulik ~
Software developer & cultural anthropologist

Tom is a software developer based in Seattle, Washington. He has lived abroad and studied foreign languages most of his life. He holds a doctorate degree in Cultural Anthropology from Brandeis University, and is in the process of making a professional transition from career academics to software engineering.

Tom has recently completed a full-stack coding bootcamp at the University of Washington, which covered JavaScript, jQuery, Express.js, React.js, Node.js, and more. Prior to that, he developed iOS apps in Objective C and Swift to train blackjack players to beat the casino. His aspirational project is to incorporate insights from sociolinguistics into natural language processing.

In his free time, Tom enjoys playing jazz and blues on the tenor saxophone, translating Spanish and Italian literature, mixing classic cocktails, and going on forest hikes and beach excursions with Ember, his black Shepherd Heeler.

  • Projective Personality Tests

    React App

    Clinical Psychometrics

    This application scores results of projective personality tests, such as the Adjective Checklist (ACL) or the Thematic Apperception Test (TAT), that a user takes. To try it out, go to the ACL page and click any of the 300 adjectives that describe your personality. When you are done, click submit. This will take you to another page that shows graph of personality dimensions from different schools of psychoanalysis that correlate to those self descriptors. The graphs for the ACL results are coded according to the specifications in the ACL Manual.

    The TAT descriptions are stored in a database. The stories are interpreted by Aylien Text Analysis - an API for Natural Language Processing (NLP). Rapid API is an alternative NLP api that focuses on emotional valence of word choice. This may be incorporated in future versions of this app.

    The app can help clinical psychologists and talk therapists manage patients’ records. It can also be used for statistical research into personality psychology.

  • Memory Test

    React App

    Memory Test

    This is a simple short-term memory game I created using React. The application renders 16 different images of inkblots to the screen, along with random captions. The user is instructed to click on each image only once. The images are shuffled after each click. The application keeps track of the user's score, which is incremented each time a different image is clicked on. The score is reset to 0 if the same image is clicked on more than once.

  • Burger Bar

    MySQL & Handlebars

    Burger Bar

    This app incorporates MySQL, Node, Express, and Handlebars, following the MVC design pattern. The user can type in the name of any specialized burger and “place an order.” The order is then added as a button in the left column. When the customer takes the burger, they click the button and the burger moves to the right column. Clicking on a burger in the right column deletes it from the database and removes if from the view.
